## Mercurial Cheat Sheet

### Create a repository
    hg init


### Check status
    hg status


### Add a file
    hg add <FILE>


### Commit
    hg commit


### Push to bitbucket
    hg push


### Pull updtes from bitbucket
    hg pull

### Delete unversioned files
    hg st -nu | xargs rm
